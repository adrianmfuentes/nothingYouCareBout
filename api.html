Juego de memoria Juego de reacción Trivial F1

¿Cómo jugar?

El objetivo del juego es responder correctamente a las preguntas sobre la Fórmula 1. Cada pregunta tiene cuatro respuestas posibles, de las cuales solo una es correcta. Para responder, selecciona la respuesta que consideres correcta. Si aciertas, sumarás un punto a tu marcador. Si fallas, no sumarás puntos.¡Buena suerte!
